# Learn Python for DevOps

This repository contains notes and references for learning Python specifically tailored for DevOps engineers. The content covers core Python fundamentals up to advanced topics like cloud automation, container orchestration, and security.

## Topics Covered

1. [Core Python Fundamentals](01-Core-Python-Fundamentals.md) - Variables, data types, loops, functions, OOP, etc.
2. [Python for Linux & System Automation](02-Python-for-Linux-System-Automation.md) - Shell commands, file management, logging, SSH.
3. [APIs & Automation](03-APIs-Automation.md) - REST APIs, requests library, HTTP methods.
4. [YAML & JSON Processing](04-YAML-JSON-Processing.md) - Handling YAML and JSON in DevOps tools.
5. [Cloud Automation (AWS/GCP/Azure)](05-Cloud-Automation.md) - boto3, cloud SDKs.
6. [Infrastructure as Code Support](06-Infrastructure-as-Code-Support.md) - Ansible, Jinja2, dynamic inventories.
7. [Docker Automation](07-Docker-Automation.md) - Docker Python library for containers.
8. [Kubernetes Automation](08-Kubernetes-Automation.md) - Kubernetes Python client.
9. [CI/CD Automation](09-CI-CD-Automation.md) - Automating pipelines with APIs.
10. [Monitoring & Observability](10-Monitoring-Observability.md) - psutil, system monitoring.
11. [Async & Concurrent Programming](11-Async-Concurrent-Programming.md) - threading, asyncio.
12. [Testing](12-Testing.md) - pytest for automation scripts.
13. [Security Automation](13-Security-Automation.md) - paramiko, cryptography.
14. [CLI Tool Development](14-CLI-Tool-Development.md) - argparse, click.
15. [Database Basics](15-Database-Basics.md) - sqlite3, SQLAlchemy.

## Getting Started

Clone the repository and start with the core fundamentals. Each file includes key concepts, code examples, and practice suggestions.

## Prerequisites

- Python 3.x
- pip for installing libraries
- Basic understanding of DevOps concepts
