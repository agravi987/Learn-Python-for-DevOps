# 09-CI-CD-Automation

## Tools APIs

- Jenkins API, GitHub Actions API, GitLab API

## Libraries

- requests: For REST API calls
- python-jenkins: `pip install python-jenkins`

## Automation

- Trigger builds: POST to Jenkins API
- Check status: GET pipeline status
- Deploy via API

## Practice

- Automate GitHub Actions: Use requests to trigger workflows
- Jenkins job management: Create/update jobs via API
