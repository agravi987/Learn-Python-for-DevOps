# 12-Testing

## pytest Library

- Install: `pip install pytest`
- Run: `pytest test_file.py`

## Writing Tests

- Functions: `def test_something(): assert condition`
- Fixtures: `@pytest.fixture def setup(): ...`

## DevOps Testing

- Test automation scripts
- Validate API responses
- Check file operations

## Practice

- Test a function: Write test for file copy script
- Mock APIs: Use pytest-mock for external calls
